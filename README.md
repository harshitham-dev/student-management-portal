# Student Management Portal

A basic web-based Java application using JSP, Servlets, and MySQL for managing student records.

## Features
- Add and view student records
- JSP-based UI
- Servlet-driven backend
- MySQL for data persistence

## Technologies
- Java
- JSP/Servlets
- Apache Tomcat
- MySQL
